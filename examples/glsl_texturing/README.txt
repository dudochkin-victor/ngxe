The Art of Texturing Using the GLSL
http://www.ozone3d.net/tutorials/glsl_texturing.php

Description:
		These demos show how to implement some of the common texturing techniques 
		in GLSL. All demos use the Hyperion Demo-System.
		Hyperion is FREE and is available here: http://www.ozone3d.net/hyperion.php
	
Controls:
	Escape Key        	- Exits the demo.
	Mouse 				- Rotate the camera
	AWSD keys			- Move the camera
