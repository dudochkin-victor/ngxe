Setting up SDL

General

SDL's web-site is: www.libsdl.org

You can download SDL there for various platvorms.

Also install the SDL-image and SDL-ttf support libraries.

Linux

If you can run sdl-config –libs and get something like:
-L/usr/lib -lSDL -lpthread
Then you SDL is correctly installed including all development files.

Linux Debian

You need the following packages: (later versions is good!)
libsdl1.2debian
libsdl1.2debian-sound (sound = oss for me)
(libsdl1.2debian-all is maybe a good idea)
libsdl-image1.2-dev
libsdl-ttf2.0-dev
libsdl1.2-dev
