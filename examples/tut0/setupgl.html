Setting up OpenGL

Linux

Firstly get the correct drivers installed for your graphics card. Nvidia drivers is downloadable from www.nvidia.com.
Radeon drivers?

Then you need to enable GLX in X. This might depend on your distro.
But in general in /usr/X11/XF86Config-4 you need:
In section “Module”: Load “glx”

To check if Your Xwindows support opengl run glxinfo from a Xterm (or something similar)
The most important line at first would be the OpenGL version.
e.g: “OpenGL version string: 1.4.1 NVIDIA 53.36”

Debian Linux

Normally packages are installed using aptitude or apt-get. You can use what you like :)

To install the Nvidia drivers you need:
nvidia-glx
nvidia-kernel-common
nvidia-kernel-*

If you want to compile the driver yourself you need: nvidia-kernel-source
Also install: nvidia-glx-dev
(Take a look at nvidia-settings)

nvidia-cg-toolkit is very handy if you don't want to install CG by hand. (CG wont be needed until tutorial 10)

For opengl development you also need:
xlibmesa-gl-dev
xlibmesa-glu-dev

You will need to install the following packages for these tutorials:
make
g++
